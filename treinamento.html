Treinamento

Ainda está sendo discutido como será feito o treinamento. O que se sabe até agora é que provavelmente iremos alocar uma sala de aula no período da tarde, e que o treino será semanal. O modo como serão feitos os treinos também estão sendo analisados, algumas das ideias é convidar professores interessados para nos dar algumas "aulas" referentes a alguns tópicos, como por exemplo o Fabio Kepler sabe bastante de programação dinâmica e poderia nos ajudar em problemas relacionados.

Entretanto, cabe aos alunos buscarem informações e trazerem problemas para serem estudados nos dias do treinamento. Por exemplo: Eu não consegui resolver determinado problema, mas cheguei perto, então levarei esse problema para ser discutido com os demais a fim de obter uma solução em conjunto. Ou, consegui resolver um problema difícil e quero compartilhar como consegui tal façanha, então levo o problema até o treinamento e lá mostrarei o problema para que o pessoal tente resolver também, no fim será comparado os métodos utilizados por cada aluno.

Além disso, serão discutidos algoritmos pouco visíveis na graduação para resolver os problemas, como por exemplo algoritmos de geometria computacional. Será visto também diferentes estruturas de dados mais avançadas, como Binary Indexed Tree, Segment Tree, Disjoint-Set, etc.

Estamos aceitando sugestões de qualquer tipo, então caso tenha alguma, nos contate.
